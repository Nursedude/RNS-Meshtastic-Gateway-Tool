# RNS Meshtastic Gateway Tool
This tool bridges the Reticulum Network Stack (RNS) with Meshtastic hardware (LoRa).

## Features
- **Reliable Interface:** Connects RNS to Meshtastic radios via USB.
- **Configurable:** Settings managed via config.py.
- **Secure:** Uses RNS encryption and Identity.

## How to Run
1. Install dependencies: \`pip install -r requirements.txt\`
2. Run as root (for USB access): \`sudo python3 main.py\`
